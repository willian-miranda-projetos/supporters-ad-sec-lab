## GPO: GPO_ADMIN_LOCAL_ATIVACAO

Objetivo:
Garantir que a conta de administrador local esteja ativa para uso com LAPS

Caminho:
Computer Configuration > Policies > Windows Settings > Security Settings > Local Policies > Security Options

Configuração:
Accounts: Administrator account status

Valor:
Enabled

---

Observação:

Essa configuração é essencial para o funcionamento do Windows LAPS.

Sem a conta de administrador local ativa:

* O LAPS não consegue gerenciar a senha
* A rotação de senha não ocorre
* O atributo msLAPS-Password não é atualizado no Active Directory

Essa GPO deve ser aplicada em todas as máquinas do domínio (OU de computadores).
